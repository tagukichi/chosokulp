# 調速 LP テーマ（chosoku-lp）

物件調査アシスタント「調速（Chosoku）」のランディングページ用 **クラシック WordPress テーマ**。
1ページ完結型で、`v2/` の静的LPをそのまま再現しています。デザイン・アニメーションは静的版と同一です。

## インストール
1. `wp-theme/chosoku-lp/` フォルダを **ZIP 圧縮**（フォルダ名は `chosoku-lp` のまま）。
2. WordPress 管理画面 → **外観 > テーマ > 新規追加 > テーマのアップロード** で ZIP を選択。
   - または `wp-content/themes/chosoku-lp/` に直接配置。
3. **有効化**。トップページにLPが表示されます（`front-page.php`）。
   - 「設定 > 表示設定」で「ホームページの表示」が固定ページの場合も、本テーマは `front-page.php` が優先され同じLPを表示します。

## 「外観 > カスタマイズ > 調速LP設定」で編集できる項目
- **ロゴ**：ヘッダー（横）／フッター（縦）。未設定なら同梱ロゴを表示。
- **「選ばれる理由」画像**：01〜06。未設定は薄グレーのプレースホルダー（`assets/img/why/placeholder.svg`）。
- **お問い合わせフォーム**：Contact Form 7 のショートコードを貼ると、その場所にCF7フォームを表示。空ならデザイン用ダミーフォーム。
- **リンク・URL**：ログイン（本体アプリ）/ デモ / プライバシーポリシー / 利用規約 / 特商法表記 / 運営会社。
- **SEO / OGP**：メタディスクリプション、OGP画像。

## Contact Form 7 連携手順
1. プラグイン「Contact Form 7」を有効化。
2. 「お問い合わせ > 新規追加」でフォームを作成（通知先メールはCF7のメールタブで設定）。
3. 発行されたショートコード（例 `[contact-form-7 id="123" title="お問い合わせ"]`）を
   **カスタマイズ > 調速LP設定 > お問い合わせフォーム** に貼り付け。
   - 同梱の `assets/css`（テーマ style.css 内）に `.contact-form` のスタイルがあるため、
     CF7 のマークアップに合わせて微調整したい場合は CSS を追記してください。

## よくある質問（FAQ）を固定ページから編集する（ACF）
FAQ は **ACF（Advanced Custom Fields・無料版）** で、ホームページに設定した固定ページの編集画面から入力できます（繰り返しフィールドは使っていません）。

セットアップ：
1. プラグイン「**Advanced Custom Fields**」（無料版）をインストール・有効化。
2. **設定 > 表示設定** で「ホームページの表示」を**固定ページ**にし、トップ用の固定ページを選択。
3. その固定ページを編集すると、**タイトル直下に「調速LP コンテンツ」**が表示されます。
4. 「よくある質問（FAQ）」タブで FAQ1〜6 の質問・回答を入力 → 更新。

仕様：
- **質問を空欄にした項目は非表示**になります（最大6件、件数はこれで調整）。
- **すべて空欄ならテーマ既定の6件**を表示します（プラグイン未導入時も同様）。
- 回答内の改行はそのまま反映されます。
- ACF が無効でもテーマは通常どおり動作します（既定のFAQを表示）。

> 機能・選ばれる理由・お悩み等のテキストも同じ方式（固定フィールド）で編集可能にできます。必要なら拡張します。

## アクセス解析（GA4）
- 「サイトキット」等のプラグイン、または子テーマ/フック（`wp_head`）で GA4 タグを追加してください。

## 構成
```
chosoku-lp/
├─ style.css            テーマヘッダー＋LP全スタイル
├─ functions.php        セットアップ・アセット読込・ヘルパー
├─ header.php           <head>／ヘッダーナビ（ロゴはカスタマイザー対応）
├─ footer.php           フッター（各URLはカスタマイザー対応）
├─ front-page.php       トップ＝LP
├─ index.php            フォールバック（同じくLP）
├─ inc/customizer.php   カスタマイザー設定
├─ template-parts/lp.php  LP本体（全セクション）
└─ assets/
   ├─ css/（※スタイルは style.css に内包）
   ├─ js/lp.js          メニュー/リビール/タイプライター 等
   └─ img/              logo-yoko.png, logo-tate.png, why/（プレースホルダー＋イラスト）
```

## 備考
- 静的版は `/v2/`（GitHub Pages 配信）に残しています。デザイン変更は静的版→本テーマの順で反映すると差分管理が楽です。
- 子テーマでのカスタマイズを推奨（本テーマ更新時に上書きされないため）。
